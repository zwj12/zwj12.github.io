# 菜单栏
如果页面需要显示菜单，首先需要启动项目的菜单栏功能，然后才能使用函数显示菜单：Project -> Properties -> Graphical design -> Runtime general -> Main menus active

# Lanaguage
标识字符串Text可以包含任何字符串，比如空格，如果需要混合语言切换和非语言切换的字符，对于语言切换的字符标识需要使用两个@包围起来。
	@Text
	@Text1@ Text2 @Text3
